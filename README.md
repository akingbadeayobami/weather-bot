# Weather Bot
> An Awesome weather bot

This project is built using

-  MongoDB - Database
-  Express - Back End
-  React - Front End
-  Wit.ai - NLP
-  Weatherbit - Weather API


###### To install the project run
```sh
npm install
```

###### To run the project

Rename `.env.default` to `.env` and get your API tokens

```
npm run dev
```

###### To run the test
```
npm run test
```
